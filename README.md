# Job challenge by Franco Josue Osorio Scheggia for the position of Junior Fullstack Developer at Nimble Gravity

## Description

This project is a frontend application built with React and Vite.
It consumes the provided API to:

- Retrieve candidate information
- Display available job positions
- Allow the user to submit a GitHub repository URL per job
- Handle loading and basic error states

## Tech Stack

- React
- Vite
- JavaScript (ES6)
- Fetch API

## Setup

1. Clone the repository
2. Install dependencies:

   npm install

3. Create a `.env` file based on `.env.example`
4. Run the development server:

   npm run dev
